# 시작하기 (Getting Started)

이 가이드에서는 OpenManus를 사용하여 첫 번째 AI 에이전트를 만들고 실행하는 방법을 단계별로 설명합니다.

## 사전 요구사항

시작하기 전에 다음 사항을 확인하세요:

- Python 3.12 이상
- uv 또는 conda (패키지 관리)
- Docker (샌드박스 실행용, 선택사항)
- LLM API 키 (OpenAI, Anthropic, Google 중 하나)

## 1단계: 설치

### uv 사용 (권장)

```bash
# uv 설치
curl -LsSf https://astral.sh/uv/install.sh | sh

# 프로젝트 클론
git clone https://github.com/openmanus/openmanus.git
cd openmanus

# 의존성 설치
uv sync
```

### conda 사용

```bash
# 가상환경 생성
conda create -n openmanus python=3.12
conda activate openmanus

# 의존성 설치
pip install -e .
```

## 2단계: API 키 설정

OpenManus는 다양한 LLM 제공업체를 지원합니다. 하나 이상의 API 키를 설정하세요.

### 환경 변수로 설정

```bash
# OpenAI
export OPENAI_API_KEY=sk-...

# Anthropic Claude
export ANTHROPIC_API_KEY=sk-ant-...

# Google Gemini
export GOOGLE_API_KEY=...
```

### 설정 파일 사용

`config.yaml` 파일을 생성하고 다음과 같이 설정합니다:

```yaml
llm:
  model: "gpt-4"  # 또는 "claude-3-5-sonnet-20241022", "gemini/gemini-1.5-pro"
  api_key: "your-api-key"  # 또는 환경 변수 사용 권장
  temperature: 0.7
  max_tokens: 4096
```

## 3단계: 첫 번째 에이전트 실행

### 간단한 작업 실행

```python
# simple_agent.py
import asyncio
from app.agent.manus import Manus

async def main():
    # 에이전트 생성
    agent = await Manus.create()
    
    try:
        # 간단한 질문
        result = await agent.run("Python으로 1부터 10까지의 합을 계산하는 코드를 작성해줘")
        print(result)
        
    finally:
        # 정리
        await agent.cleanup()

if __name__ == "__main__":
    asyncio.run(main())
```

실행:

```bash
python simple_agent.py
```

### 명령줄에서 실행

```bash
# 직접 프롬프트 전달
python main.py --prompt "현재 시간을 알려주고, 5초 후의 시간을 계산해줘"

# 대화형 모드
python main.py
> 안녕? 오늘 할 일을 정리해줄 수 있어?
```

## 5단계: 복잡한 작업 수행

### 데이터 분석 예제

```python
async def main():
    agent = await Manus.create()
    
    try:
        result = await agent.run("""
        다음 작업을 순서대로 수행해줘:
        
        1. CSV 데이터 생성:
           - 100개의 랜덤한 판매 데이터
           - 컬럼: date, product, quantity, price
           - sales_data.csv로 저장
        
        2. 데이터 분석:
           - 제품별 총 매출 계산
           - 일별 판매 추이 분석
           - 기술 통계 계산
        
        3. 시각화:
           - 제품별 매출 막대 차트
           - 일별 판매량 꺾은선 그래프
           - 결과를 analysis.png로 저장
        
        4. 리포트 작성:
           - 분석 결과 요약
           - 주요 인사이트
           - report.md 파일로 저장
        """)
        print(result)
        
    finally:
        await agent.cleanup()

asyncio.run(main())
```

## 문제 해결

### API 키 오류

```
Error: API key not found
```

**해결**: 환경 변수를 올바르게 설정했는지 확인하세요.

```bash
echo $OPENAI_API_KEY  # 키가 출력되어야 함
```

### 샌드박스 연결 실패

```
Error: Cannot connect to sandbox
```

**해결**: Docker가 실행 중인지 확인하세요.

```bash
docker ps  # Docker가 실행 중이어야 함
```

### 패키지 설치 문제

```
Error: No module named 'app'
```

**해결**: 프로젝트를 올바르게 설치했는지 확인하세요.

```bash
pip install -e .  # 또는 uv sync
```

## 팁과 요령

### 1. 명확한 지시사항

에이전트에게 구체적이고 명확한 지시를 제공하세요:

```python
# ❌ 모호한 지시
await agent.run("데이터 분석해줘")

# ✅ 명확한 지시
await agent.run("""
sales_data.csv 파일을 읽어서:
1. 월별 매출 합계 계산
2. 전월 대비 증감률 계산
3. 결과를 막대 차트로 시각화
""")
```

### 2. 단계별 작업 분리

복잡한 작업은 여러 단계로 나누세요:

```python
# 1단계: 데이터 수집
await agent.run("웹사이트에서 데이터 수집")

# 2단계: 데이터 처리
await agent.run("수집한 데이터 정제 및 변환")

# 3단계: 분석
await agent.run("처리된 데이터 분석")
```

### 3. 결과 검증

중요한 작업 후에는 결과를 검증하세요:

```python
result = await agent.run("""
파일을 수정한 후:
1. 문법 오류 확인
2. 테스트 실행
3. 결과 리포트
""")
```

### 4. 에러 처리

예외 처리를 포함하세요:

```python
try:
    result = await agent.run("작업 수행")
    print(f"성공: {result}")
except Exception as e:
    print(f"오류 발생: {e}")
finally:
    await agent.cleanup()
```

## 예제 프로젝트

### 일일 뉴스 요약봇

```python
# news_summarizer.py
import asyncio
from datetime import datetime
from app.agent.manus import Manus

async def daily_news_summary():
    agent = await Manus.create()
    
    try:
        today = datetime.now().strftime("%Y-%m-%d")
        
        result = await agent.run(f"""
        {today} 일일 뉴스 요약:
        
        1. Hacker News에서 상위 10개 기사 수집
        2. 각 기사 제목과 요약 작성
        3. 카테고리별로 분류 (기술/비즈니스/과학 등)
        4. Markdown 리포트 생성 (news_{today}.md)
        5. 중요도 순으로 정렬
        """)
        
        print(f"뉴스 요약 완료: {result}")
        
    finally:
        await agent.cleanup()

if __name__ == "__main__":
    asyncio.run(daily_news_summary())
```
