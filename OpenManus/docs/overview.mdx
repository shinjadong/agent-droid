# OpenManus

OpenManus는 다양한 작업을 자동화할 수 있는 **범용 AI 에이전트 프레임워크**입니다. 초대 코드 없이 누구나 자유롭게 사용할 수 있으며, 브라우저 자동화, 코드 실행, 파일 편집, 웹 검색 등 다양한 기능을 제공합니다.

📖 **범용 AI 에이전트(General AI Agent)**
├─ **정의**: 특정 작업에 국한되지 않고 다양한 종류의 작업을 자동으로 수행할 수 있는 인공지능 시스템
├─ **쉬운 비유**: 만능 비서처럼 문서 작성부터 데이터 분석, 웹 탐색까지 다양한 업무를 처리
├─ **메커니즘**: LLM이 주어진 도구들을 선택하고 조합하여 복잡한 작업을 단계별로 해결
└─ **존재 이유**: 반복적이고 시간 소모적인 작업을 자동화하여 생산성 향상

## 주요 특징

### 🛠️ 다양한 도구 지원

OpenManus는 다음과 같은 강력한 도구들을 기본 제공합니다:

- **Python 실행**: 데이터 분석, 계산, 파일 처리 등을 위한 Python 코드 실행
- **브라우저 자동화**: Playwright 기반의 웹 브라우저 제어 및 자동화
- **파일 편집**: 코드 파일의 문자열 검색 및 대체를 통한 정밀한 편집
- **웹 검색**: Google, DuckDuckGo, Bing 등 다양한 검색 엔진 지원
- **사용자 대화**: 추가 정보가 필요할 때 사용자에게 질문

### 🔌 MCP (Model Context Protocol) 통합

📖 **MCP (Model Context Protocol)**
├─ **정의**: AI 모델이 외부 도구와 리소스에 접근할 수 있도록 하는 표준 프로토콜
├─ **쉬운 비유**: 플러그인 시스템처럼 필요한 기능을 모듈처럼 추가/제거
├─ **메커니즘**: SSE 또는 STDIO 방식으로 MCP 서버에 연결하여 도구 동적 로드
└─ **존재 이유**: 에이전트 기능을 코드 수정 없이 확장 가능하게 만듦

OpenManus는 MCP를 지원하여 외부 도구를 동적으로 연결할 수 있습니다:

```toml
# config/mcp.json 예시
{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/workspace"]
    }
  }
}
```

### 🤖 다중 에이전트 워크플로우

복잡한 작업을 위해 여러 에이전트가 협력할 수 있습니다:

- **Manus**: 범용 작업 처리 에이전트
- **DataAnalysis**: 데이터 분석 및 시각화 전문 에이전트
- **Planning**: 복잡한 작업을 분해하고 계획하는 코디네이터

### 🐳 샌드박스 실행 환경

📖 **샌드박스(Sandbox)**
├─ **정의**: 시스템에 영향을 주지 않고 코드를 안전하게 실행할 수 있는 격리된 환경
├─ **쉬운 비유**: 실험실의 격리 부스처럼 위험한 실험도 안전하게 수행
├─ **메커니즘**: Docker 컨테이너를 사용하여 메모리, CPU, 네트워크 접근 제한
└─ **존재 이유**: 신뢰할 수 없는 코드 실행 시 시스템 보안 유지

```toml
[sandbox]
use_sandbox = true
image = "python:3.12-slim"
memory_limit = "512m"
cpu_limit = 1.0
timeout = 300
network_enabled = false
```

## 사용 사례

### 💼 업무 자동화

- 반복적인 보고서 생성 및 데이터 정리
- 웹사이트에서 정보 수집 및 정리
- 파일 일괄 처리 및 변환

### 📊 데이터 분석

- CSV, Excel 파일 분석 및 시각화
- 통계 계산 및 그래프 생성
- 데이터 정제 및 변환

### 🌐 웹 작업

- 웹 스크래핑 및 모니터링
- 양식 자동 입력 및 제출
- 브라우저 기반 테스트 자동화

### 💻 개발 지원

- 코드 리팩토링 및 포맷팅
- 문서 생성 및 업데이트
- 프로젝트 설정 및 구성

## 시작하기

OpenManus를 사용하려면 다음 단계를 따르세요:

1. **설치**: Python 3.12+ 환경에서 의존성 설치
2. **설정**: API 키 및 에이전트 설정 구성
3. **실행**: 원하는 모드로 에이전트 실행

자세한 내용은 [빠른 시작 가이드](/quickstart)를 참조하세요.
