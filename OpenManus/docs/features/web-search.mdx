# 웹 검색 (Web Search)

OpenManus는 웹 검색 도구를 통해 실시간 정보를 수집하고 활용할 수 있습니다. 최신 뉴스, 기술 문서, 제품 정보 등 다양한 웹 리소스에 접근하여 에이전트가 최신 정보를 기반으로 작업을 수행할 수 있습니다.

## 개요

웹 검색 기능의 주요 특징:

- **실시간 정보 검색**: 최신 웹 콘텐츠 접근
- **다양한 검색 엔진**: DuckDuckGo, Google, Bing 등 지원
- **스마트 결과 필터링**: 관련성 높은 결과 선택
- **콘텐츠 추출**: 웹 페이지에서 핵심 정보 자동 추출
- **멀티 쿼리**: 여러 검색을 조합하여 포괄적 정보 수집

## 기본 사용법

### 간단한 검색

```python
from app.agent.manus import Manus

agent = await Manus.create()

# 기본 웹 검색
await agent.run("""
'OpenAI GPT-4 최신 뉴스'를 웹에서 검색해서
주요 내용을 요약해줘.
""")

# 특정 주제 검색
await agent.run("""
Python 3.12의 새로운 기능을 검색하고
주요 변경사항을 정리해줘.
""")
```

### 정보 수집 및 정리

```python
# 여러 소스에서 정보 수집
await agent.run("""
'머신러닝 최적화 기법'에 대해:
1. 웹 검색으로 최신 정보 수집
2. 상위 5개 결과 요약
3. 핵심 개념 정리
4. 참고 링크 포함
""")

# 비교 분석
await agent.run("""
'React'와 'Vue.js'를 비교 검색해서:
1. 각각의 장단점
2. 인기도 트렌드
3. 사용 사례
4. 선택 가이드
""")
```

## 고급 검색 기법

### 시간 기반 검색

```python
# 최신 뉴스 검색
await agent.run("""
최근 24시간 이내의 'AI 기술' 관련 뉴스를 검색하고
주요 트렌드를 분석해줘.
""")

# 특정 기간 검색
await agent.run("""
2024년 1월부터 현재까지의
'암호화폐 시장' 동향을 검색하고 정리해줘.
""")
```

### 특정 사이트 검색

```python
# 특정 도메인에서만 검색
await agent.run("""
GitHub에서 'python data analysis'
관련 인기 저장소를 검색해줘.
사이트: github.com
""")

# 기술 문서 검색
await agent.run("""
공식 문서에서 'Django REST framework'
튜토리얼을 찾아줘.
사이트: djangoproject.com
""")
```

### 이미지 및 비디오 검색

```python
# 이미지 검색
await agent.run("""
'데이터 시각화 예제' 이미지를 검색하고
가장 유용한 차트 5개를 다운로드해줘.
""")

# 비디오 검색
await agent.run("""
YouTube에서 'Python pandas tutorial'
관련 동영상을 찾고 상위 3개의
제목과 링크를 알려줘.
""")
```

## 검색 결과 활용

### 데이터 수집 및 저장

```python
await agent.run("""
'인공지능 스타트업'을 검색해서:
1. 상위 10개 회사 목록
2. 각 회사의 주요 제품
3. 설립 연도 및 투자 정보
결과를 CSV 파일로 저장
""")
```

### 리서치 보고서 작성

```python
await agent.run("""
'양자 컴퓨팅' 주제로 리서치 보고서 작성:
1. 웹 검색으로 최신 논문 및 기사 수집
2. 핵심 개념 정리
3. 현재 기술 수준
4. 미래 전망
5. 참고 자료 목록
마크다운 형식으로 보고서 작성
""")
```

### 경쟁사 분석

```python
await agent.run("""
경쟁사 'Company X' 분석:
1. 최신 뉴스 및 보도자료
2. 주요 제품 및 서비스
3. 고객 리뷰 및 평가
4. 소셜 미디어 반응
결과를 종합 리포트로 작성
""")
```

## 모범 사례

### 1. 효과적인 검색 쿼리

```python
# ❌ 모호한 쿼리
await agent.run("AI 검색")

# ✅ 구체적인 쿼리
await agent.run("""
'GPT-4 API 사용법 및 모범 사례'를
공식 문서 및 기술 블로그에서 검색
""")
```

### 2. 결과 검증

```python
# ✅ 여러 소스 확인
await agent.run("""
'Python 보안 취약점'을 검색할 때:
1. 공식 Python 보안 공지
2. 주요 보안 뉴스 사이트
3. GitHub 보안 권고
모든 소스를 종합하여 검증된 정보만 제공
""")
```

### 3. 최신성 확인

```python
# ✅ 발행일 확인
await agent.run("""
검색 결과에서 각 정보의 발행일을 확인하고
1년 이내의 최신 정보만 사용해줘.
""")
```

## 문제 해결

### 검색 결과가 없는 경우

```python
# 대체 검색어 시도
await agent.run("""
'특정 기술 용어'로 검색:
1. 영문 키워드로 재검색
2. 유사 용어로 검색
3. 더 일반적인 용어로 확장 검색
""")
```

### 부정확한 정보 필터링

```python
# 신뢰할 수 있는 소스 우선
await agent.run("""
검색시 다음 사이트를 우선 확인:
- 공식 문서 사이트
- .edu, .gov 도메인
- 유명 기술 블로그
- 학술 논문
""")
```
